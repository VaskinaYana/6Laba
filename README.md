### Алгоритм решения

1. Факториал числа:
Если число равно 0, факториал = 1
Иначе факториал = число × факториал (число − 1)
2. Сумма факториалов нечётных чисел от N до Max:
Если N > Max → сумма = 0 (остановка)
Если N чётное → пропускаем N, считаем сумму для N+1
Если N нечётное → берём факториал N, прибавляем сумму для N+1

Необходимо реализовать определение класса (поля, свойства, конструкторы, перегрузка

может быть меньше 0). Результат должен быть типа Money ). Протестировать все методы, включая конструкторы.

### Алгоритм решения

1.Если список пустой → смен знака = 0
2.Если в списке один элемент → смен знака = 0
3.Если в списке два или более элемента:
Берём первые два числа X и Y
Если X положительный и Y отрицательный → это смена знака (+1), дальше считаем смены для списка, начинающегося с Y
Если X отрицательный и Y положительный → это смена знака (+1), дальше считаем смены для списка, начинающегося с Y
Если знаки одинаковые (оба + или оба -) → смены знака нет (+0), дальше считаем смены для списка, начинающегося с Y
4.Складываем все единицы, полученные на каждом шаге
